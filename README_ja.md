# Conti Neue 2P Readme

## 概要

「Conti Neue 2P」は、「Press Start 2P」と「美咲ゴシック第2」を基に、オリジナルのひらがな・カタカナを追加した改変フォントです。
レトロなビデオゲームの雰囲気を持つビットマップ風フォントでありながら、日本語の読みやすさも考慮してデザインしました。
